ESTRUTURAS DE REPETIÇÃO EM C

As estruturas de repetição, executam a repetição de um conjunto de instruções enquanto uma determinada condição é verdadeira. Na linguagem de programação C, existem três estruturas de repetição, são elas: do while, while e for.

Cada uma destas estruturas tem a sua particularidade em termos de funcionamento. E mostrarei-as abaixo:

DO WHILE

Esta estrutura de repetição, garante que o bloco de instruções seja executado no mínimo uma vez, já que a condição que controla o laço é testada apenas no final do comando.

No do while o bloco de comandos é executado pelo menos uma vez de forma obrigatória, independente do resultado da expressão lógica.

É composto por:

Comando: pode ser um bloco de comando vazio ou de repetição definido por um término.

Condição: Pode ser qualquer expressão valida ou até quando a verdadeira. 
 
    int main(){

     float nota1=0,nota2=0,media=0;
     int resp;
                                    
     do{

      printf("Digite a primeira nota: ");
      scanf("%f",¬a1);
      printf("Digite a segunda nota: ");

      media = (nota1 + nota2)/2;
      printf("Media do aluno = %f\n",media);

      scanf("%d", &resp);

WHILE

Nesse loop de repetição pode existir uma condição de término definida já no inicio ou não.

A diferença entre o comando while e o do while é justamente o local onde a condição que controla o laço é testada.

Tendo em vista que no comando while a condição é testada antes do bloco de instruções, e caso a condição seja falsa a repetição não será executada e o código vai para a próxima instrução fora do laço..

É composto por:

Condição: Pode ser qualquer expressão valida ou até quando a verdadeira.

Comando: pode ser um bloco de comando vazio ou de repetição definido por um término. 

            
    int main(){

     int contador = 1;
    
     while (contador <= 10) 

     {
        printf("%d ", contador);
        contador++; 
     }  
    
     return 0;
    }

FOR

Esse loop é utilizado geralmente quando existe um termino definido, sempre vai tem um fim definido para esse loop implementado no inicio.

Já que é composto por:

Variável de inicialização: comando de atribuição que inicia uma variável que faz o controle do laço.

Condição: expressão usada que determinará o final do laço.

Incremento: define a variável de controle e muda a cada passada no laço. 

                        
int main(){

  int contador;
  
  for (contador = 10; contador >= 1; contador--){
    
    printf("%d ", contador);
  }
  
  return 0;
}
